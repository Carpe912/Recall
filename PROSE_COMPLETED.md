## 🚀 如何使用

### 启动项目

```bash
# 1. 确保使用 Node.js 22
nvm use 22

# 2. 安装依赖
pnpm install

# 3. 启动开发服务器
pnpm dev
```

### 访问页面

- **图形编辑器**: http://localhost:3000/
- **Prose 编辑器**: http://localhost:3000/prose.html

### 页面导航

- 在图形编辑器页面，点击右上角 **"查看 Prose 编辑器 →"**
- 在 Prose 编辑器页面，点击右上角 **"← 返回图形编辑器"**

## 🎨 Prose 编辑器功能

### 基础功能
- 粗体、斜体、下划线、删除线
- H1-H6 标题
- 文本对齐（左、中、右）
- 无序列表、有序列表、任务列表
- 引用块、代码块、分割线
- 链接、图片、表格
- 文字颜色、背景高亮
- 撤销/重做

### 自定义功能 ⭐
- **分栏布局**: 点击 📊 按钮插入 2 栏或 3 栏
- **高亮块**: 点击对应按钮插入
  - 📘 信息块（蓝色）
  - ⚠️ 警告块（黄色）
  - ✅ 成功块（绿色）
  - ❌ 错误块（红色）
  - 📝 笔记块（灰色）
